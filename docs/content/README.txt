File extension conventions
==========================

* Web site pages
    - content.md
    - data.md
    - engine.md
    - home.md
    - structure.md

* Partial MarkDown to be included
    - EXAMPLE.txt
    - FLOW.txt
    - LINKS.txt
    - opt/*.txt

* MarkDown snippets
    - snippets.yaml

* Other unrelated MarkDown files
    - help.markdown
    - jqt.1.markdown
