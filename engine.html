jq templates

The jq template language will be called jqt. The tools used in the implementation of jqt are:

  • jq, a lightweight and flexible command-line JSON processor.
  • Bash, sed and other shell tools.
  • GNU Make, the tool that builds executable programs and libraries from source code.
  • GPP, a general-purpose preprocessor.
  • Pandoc, a universal document converter.
